# Joblock 求职助手

> 在招聘网站中屏蔽企业

1. 支持多种名单管理：黑名单、白名单、低关注名单
2. 自动计算年薪，以及依据年薪过滤
3. 带GUI界面，可以即时导出，导入列表名单
4. 支持Boss直聘、拉勾网、猎聘网
